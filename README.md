## 🧸 Cyworld Clone Project

싸이월드의 대표적인 레트로 감성을 재현한 정적 웹 프로젝트입니다.  
HTML, CSS, JavaScript를 이용해 싸이월드 스타일의 UI와 레이아웃을 구현하는 데 집중했습니다.

---

### 🛠 기술 스택

- HTML5  
- CSS3 (Flexbox, Grid 포함)  
- JavaScript (Vanilla)  

---

### 📁 주요 내용

- 싸이월드 메인 화면 레이아웃 및 스타일링  
- 메뉴 인터랙션 (열기/닫기)  
- 반응형 웹 기본 구성  
- 정적 웹사이트 구조 설계 및 구현
